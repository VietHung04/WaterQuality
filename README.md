# Water Quality Analysis - Data Mining Project

## Overview
This project analyzes water quality using data mining techniques.

## Structure
- data/: dataset
- notebooks/: analysis notebooks
- src/: source code
- models/: trained models
- report/: final report

## Methods
- EDA
- Preprocessing
- Association Rules
- Clustering
- Classification
- Semi-supervised learning

## Best Model
Random Forest achieved the best performance.
